# Data Cleaning

- scripts for cleaning Eugene CAD data and SPR calls for service data

# Contents

## Eugene data

EUG_data_cleaning.ipynb: loads the raw data (11 csv files) and outputs one cleaned file (welfare.csv) to “../data/“

### Input Files

located in “../data/Eugene_CAD_data_noloc“:

EugeneCAD2015noloc.csv

EugeneCAD2016noloc.csv

EugeneCAD2017noloc.csv

EugeneCAD2018noloc.csv

EugeneCAD2019noloc.csv

EugeneCAD2020noloc.csv

EugeneCAD2021noloc.csv

EugeneCAD2022noloc.csv

EugeneCAD2023noloc.csv

EugeneCAD2024noloc.csv

EugeneCAD2025noloc.csv


## Springfield data

SPR_data_clearning.ipynb: leads the raw data (3 Excel files) and outputs one cleaned file (SPR_welfare.csv) to ''../data/"

### Input files 

located in "../data":

2015-2025 SPD Calls for Service.xlsx

2015-2025 SPD Calls with Close Codes.xlsx

2015-2025 SPD Responding Units.xlsx
