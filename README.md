# Welfare Check Response Analysis
A project for DSCI 410 - Data Science in Action. Professor: Dr. Rori Rohlfs

This project is intended to answer questions about how emergency responses to calls for welfare checks in Eugene, Oregon, have changed since the termination of CAHOOTS crisis intervention services in the city. Specifically, 

 - How does the total number of welfare checks called in per month since the stoppage compare to the number called in when CAHOOTS was operating? 
    
  - How does the number of welfare checks handled in Eugene per month by the Eugene Police Department (EPD) since the stoppage compare to the number they handled monthly when CAHOOTS was operating, and to the number handled by CAHOOTS when they were operating? 
    
  - Are the response times of EPD vs. CAHOOTS for welfare checks different, and how has the CAHOOTS stoppage affected EPD welfare check response times if at all?
    
  - Is there a statistically significant difference in the outcomes (‘closed_as’) of the welfare checks taken by CAHOOTS vs. EPD?

The repository is organized into cleaning and analysis sections. For more information, see the README files in each section.
